# Changelog

## 0.1.0 — 2026-04-23

Initial release.

- Lexer and parser for the Agnostic language (`.agn` files).
- Five built-in lint rules: `unused-var`, `unused-import`, `unreachable-code`, `missing-return`, `empty-block`.
- ANSI-colored diagnostic output with source line and caret display.
- CLI flags: `--disable`, `--enable`, `--Werror`, `--no-color`, `--no-caret`, `--list-rules`.
